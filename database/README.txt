🚀 快速开始（3步完成）
================================================================================

第1步：选择你的操作系统
  - Windows用户 → 进入 windows 文件夹
  - Linux用户   → 进入 linux 文件夹

第2步：运行部署脚本
  - Windows: 双击 "一键部署.bat"
  - Linux: 执行 ./一键部署.sh

第3步：输入MySQL信息
  - 输入MySQL用户名（通常是 root）
  - 输入MySQL密码
  - 等待完成

就这么简单！
